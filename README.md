# Faqora FAQ Schema Blocks

Faqora FAQ Schema Blocks is a lightweight WordPress plugin for creating reusable FAQ sections with a shortcode and automatically outputting valid JSON-LD FAQPage schema markup.

The plugin is built for website owners, SEO specialists, content teams, agencies, and WordPress developers who need a simple way to add structured FAQ content to pages, posts, landing pages, and service pages.

## Features

* Create reusable FAQ blocks in the WordPress admin
* Display FAQ sections with a simple shortcode
* Automatically generate valid JSON-LD FAQPage schema
* Optional accordion-style layout
* Custom heading support
* Lightweight frontend CSS
* No external API requests
* No tracking
* No hidden frontend links
* Translation-ready

## Shortcode usage

Basic usage:

```text
[faq_schema_block id="123"]
```

With a custom title:

```text
[faq_schema_block id="123" title="Frequently Asked Questions"]
```

With accordion mode:

```text
[faq_schema_block id="123" accordion="true"]
```

Disable schema for a specific block:

```text
[faq_schema_block id="123" schema="false"]
```

## Why use FAQ schema?

FAQPage schema helps search engines better understand question-and-answer content on your website. It can be useful for SEO, structured data implementation, content optimization, and AI-ready answer formatting.
